# ADR-0051: Documentation Reconciliation

Status: Accepted

Documentation is organized around user journeys and optional composition rather than one vertical slice. Guides cover producer-only streams, external producers, consumers of existing streams, CDC-only, events-only, lineage controls, lineage admission, and production Compose.

Generated bundle docs summarize graph state, bindings, external resources, policy decisions, overrides, verification, recovery, provenance, and checksums.
